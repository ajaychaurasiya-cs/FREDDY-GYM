## Run Locally

Clone the project

```bash
  git clone https://github.com/ajaychaurasiya-cs/FREDDY-GYM
```

Go to the project directory

```bash
  cd FREDDY-GYM
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
